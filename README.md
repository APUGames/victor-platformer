## ğŸš€ Getting Started

### Prerequisites
- Unity Hub
- Unity Editor (2020.3 LTS or newer recommended)
- Git for version control

### Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/APUGames/victor-platformer.git
cd victor-platformer
```

2. **Open in Unity**
   - Launch Unity Hub
   - Click "Add" and select the cloned project folder
   - Open the project with a compatible Unity version
   - Wait for Unity to import all assets

3. **Run the game**
   - Open the main game scene (usually in `Assets/Scenes/`)
   - Click the Play button in the Unity Editor
   - Test the gameplay and controls

### Controls

- **Arrow Keys / A-D**: Move left and right
- **Space / W / Up Arrow**: Jump
- **ESC**: Pause (if implemented)
